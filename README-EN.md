# 🔫 Duel-Game

The ideal dueling game to take on a friend!

## 📚 Installation
1. Download the launcher “[Launcher Duel Game.exe](https://github.com/loicsmith/Duel-Game/releases/latest)”
2. Place the launcher in a folder
3. Launch the launcher
4. Have fun!
