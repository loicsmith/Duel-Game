# 🔫 Duel-Game

Le jeu de duel idéal pour affronter un ami !

## 📚 Installation
1. Télécharger le jeu "[Duel Game.zip](https://github.com/loicsmith/Duel-Game/releases/latest)"
2. Extraire le .zip où vous souhaitez
3. Lancer le .exe
4. Have fun !
